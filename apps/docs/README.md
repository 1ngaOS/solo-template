# Documentation Site

Docusaurus-based documentation site for the Solo Monorepo Template.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Serve production build locally
pnpm serve
```

## Documentation Structure

- `docs/` - Documentation markdown files
- `src/pages/` - Additional pages
- `blog/` - Blog posts (optional)

## Adding Documentation

1. Add markdown files to `docs/`
2. Update `sidebars.ts` to include new pages
3. Documentation will be automatically built and served
